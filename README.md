# NASA APOD Dashboard

A minimalist, obsidian-dark dashboard that fetches daily cosmic photography and data directly from the NASA API.

## Live Demo

[View Live Project](https://iaintsurya.github.io/NASA-APOD/)

## Features

* **Daily Cosmic Sync:** Dynamically pulls the latest Astronomy Picture of the Day.
* **Obsidian Theme:** Clean, dark-mode user interface designed for visual focus.
* **Secure Architecture:** Built using environment pipelines to prevent credential leaks.

## Stack

* **Frontend:** Vanilla JavaScript, CSS3
* **Tooling:** Vite
* **Deployment:** GitHub Actions
